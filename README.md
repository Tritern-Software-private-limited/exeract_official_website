# deployment Guide

## Overview Pre-req

1) npm install
2) npm install netlify-cli
3) npm install mongodb
4) npm install bcryptjs


## Guide to run

1) netlify dev

## Deployment Guid

- go to netlify and create a new project.
- connect you github with nelify selecting only the repository  exeract_offical_website from the list.
- then click deploy.
- copy past .env file from the repository in the netlify environmental varibale.


## site is now live

(note: mongodb connection url= should be production db url not localhost one)
